Support

How to get help, report bugs and send feedback.

Report a bug / send feedback

The fastest way is inside the app: Settings → Send feedback / report a bug. Reports include your app version, an anonymous hardware summary and (optionally) recent log lines - everything needed to actually fix the issue.

Common questions

A game won't launch from the library

  • Games with a rocket badge launch through their own launcher (Steam, Epic, Riot...). Make sure that launcher is installed and signed in.
  • For unrecognized games, use Games → Add game and point at the real game exe.

The FPS overlay shows "-- FPS"

  • PresentMon must be configured once: System → FPS Overlay → Locate exe.
  • Run Nitro Forge as Administrator (frame timing uses Windows ETW, which needs elevation).
  • Games in exclusive fullscreen bypass all desktop overlays - use borderless/windowed.

Some boost options are skipped

Pausing Windows services requires Administrator rights. Everything else works without elevation and is logged in Insights → Logs.

Where is my data?

Everything lives in %LOCALAPPDATA%\NitroForge - settings, performance history, logs and crash reports. Delete that folder and the app is factory-reset.

Safety promise: every change Nitro Forge makes is recorded and reverted - on undo, on game exit, on app close, and via a backstop if the app crashes. Antivirus/security software is hard-blocked in code and can never be touched, by any setting.
